# AI Agent Rules

1.  **Semantic Styling**: Always use semantic Tailwind classes (e.g., `bg-background`, `text-foreground`, `border-border`, `bg-primary`) instead of hardcoded colors (e.g., `bg-white`, `text-black`, `border-gray-200`). This ensures consistency and proper theming support (dark/light mode).
2.  **No Custom Color Names**: Do not invent new color names or use arbitrary values if a semantic alternative exists in the design system.
